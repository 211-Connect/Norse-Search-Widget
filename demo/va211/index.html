üìù Basic Implementation

Here's the widget with default configuration. The widget automatically fetches its configuration from CMS (Payload) based on the provided tenant ID.

HTML Code:

<!DOCTYPE html>
<html>
  <head>
    <!-- 1. Include the widget stylesheet in <head> -->
    <link rel="stylesheet" href="https://search-widget.c211.io/latest/search-widget.css" />
  </head>
  <body>
    <!-- 2. Add the widget container in <body> where you want it to appear -->
    <div
      id="search-widget"
      tid="06f86a2b-c060-4374-8aca-3d70b280cb24"
      d="https://search.211virginia.org"
      l="en"
    ></div>

    <!-- 3. Load the widget script at the end of <body> -->
    <script src="https://search-widget.c211.io/latest/search-widget.umd.js"></script>
  </body>
</html>

Configuration Attributes:

  • id - Widget ID: Must be "search-widget". This determines where the widget is embedded on your page.
  • tid - Tenant ID: Unique identifier for your organization (fetches config from Payload CMS)
  • d - Domain: API domain for your 211 service
  • l - Locale: Language code (en, es, etc.) - defaults to "en"
